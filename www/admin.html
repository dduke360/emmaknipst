Case Studies

Create dynamic story pages without touching the connected photo records. Stories are stored in `settings.case_studies` and rendered via `?story=slug`.

Story Setup Guide

Follow these steps to create a complete case study that appears correctly on the public site.

  1. 1
    Create the draft Click `New Story`, then add a clear title. The slug is generated automatically and can be edited.
  2. 2
    Fill key copy fields Add `Category Label`, `Intro`, and `Story` text. Keep intro short and use story text for the narrative.
  3. 3
    Select story photos Choose all images in `Story Photos`, then set one as `Featured Photo` for the large hero frame.
  4. 4
    Set status and save Use `Draft` while editing. Switch to `Published` and click `Save Story` when ready.
  5. 5
    Preview and verify Use `Open Preview` to confirm layout, image order, and the URL `/?story=your-slug`.
